Projects

I built Qbert Cubes to challenge my CSS abilities. The project is built using CSS and HTML almost exclusively with the exception of a few lines of javascript to handle character movement. I've enjoyed building this project and time willing will be able to make it into a fully playable game. You can find the source code on GitHub Qbert-CSS

Technologies used

I used Visual Studio code, HTML, CSS, and vanilla javascript.

Built in Unity 3D in 2012 for Android and iOS. This project was built with a friend and released on Google Play and the App Store and was downloaded over thirty thousand times. I was responsible for creating sound, game assets, backgrounds, level design, menu design and code.
